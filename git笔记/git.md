# 仓库

存储项目的仓库，可能是一个项目

仓库分为本地仓库和云端仓库

# 本地区域：

分为工作区，暂存区和完成区

# 新增和修改

```
git add .

git commit -m "wuguo31"
```

# 删除

```
git rm <file>

git commit -m "wuguo31"
```



# 查看本地查看

git status

# 查看账号信息

git config --list



# 复制远程项目

git clone <url>

# 本地仓库同步到远程仓库

git push
